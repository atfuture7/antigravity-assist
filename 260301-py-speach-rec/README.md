# Simple Speech Recognition Tool

A simple command-line tool for speech recognition using Python.

## Prerequisites

- Python 3.6 or higher
- `pydub` library
- `SpeechRecognition` library
- `ffmpeg` (for audio processing)

## Instructions

How to use:

```bash
python speech_rec.py -i <input_file> -l <language> -o <output_file>
```
